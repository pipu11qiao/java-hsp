# 第四章 运算符

几乎所有运算符只能操作基本类型。但是=、==、!=可以操作所有对象。除此外String类支持+和+=

对对象赋值操作，会产生别名。=号两侧都是对应的别名

## 运算符

一元运算符 +-*/ %
关系运算符 == !=
逻辑运算符 && || ！
短路 condition1 && condition2 节省部分不必要的代码
字面值常量 需要声明类型
下划线 用来在视觉上阅读大数
指数计数法 1.39#-43f
位运算符 << >> >>> <<< | & ^
三元运算法 condition ? a : b
字符串运算符 x + " " + y
